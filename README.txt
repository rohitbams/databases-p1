Welcome to Game-Rating-System:
This application is a repository of various games that users can add and give them rating based on how much they like the game!
 
Based on individual ratings of several users, this application calculates an average rating score that shows how well a game is liked by the community.
 
You can carry out several operations in this application, like add user, add and rate games, list all games, list all users, etc.
